消息发送样例

1、基本例子

先导入maven依赖

1.1 发送消息

发送消息的步骤：

  1. 创建消息生产者producer，并制定生产者组名
  2. 指定Nameserver地址
  3. 启动producer
  4. 创建消息对象，指定主体Topic、Tag和消息体
  5. 发送消息
  6. 关闭生产者producer
1.1.1 发送同步消息

同步消息，在上一条消息发送完后，才发送下一条

1.1.2 发送异步消息

异步消息，不用判断上一条消息是否发送完毕，就发送下一条消息

1.1.3 发送单向消息

发送单向消息不需要返回结果，如日志发送等

发送完消息后，可以在rocketmq-dashboard看发送的消息

1.2 消费消息

消费消息的步骤：

  1. 创建消费者Consumer，制定消费者组名
  2. 指定Nameserver地址
  3. 订阅主题Topic和Tag
  4. 设置回调函数，处理消息
  5. 启动消费者Consumer
